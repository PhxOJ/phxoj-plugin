{{_("Few common reasons for a compile error are as follows:")}}

  • {{_("1. Incorrect selection of programming language for the submission")}}
  • {{_("2. Not using 'Main' as the main class name in Java")}}

{{_("Important notes for C/C++ users:")}}

  • {{_("1. '__int64' should be written as 'long long' in GNU C++")}}
  • {{_("2. The return type of the 'main()' function must be 'int', not void'.")}}
  • {{_("3. In 'for' statements like 'for (int i = 0; ...) {...}', the index variable 'i' becomes invalid after the loop.")}}
  • {{_("4. 'itoa' is not a standard ANSI function; hence, is not available in standard C/C++ (same goes with other non-standard ANSI functions).")}}
  • {{_("5. Using the '%lf' format specifier in 'printf' is incorrect.")}}
